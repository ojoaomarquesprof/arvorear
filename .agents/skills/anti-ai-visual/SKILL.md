# Anti-AI Visual

Quando esta skill for usada, revise o frontend para remover aparência genérica de IA.

Procure e corrija:

- hero genérico;
- cards repetitivos;
- gradientes exagerados;
- copy vaga;
- layout com cara de template;
- seções previsíveis;
- excesso de ícones;
- espaçamentos artificiais;
- falta de identidade própria;
- falta de detalhes reais do negócio.

O resultado deve parecer feito sob medida, com aparência premium, humana, profissional e pronta para apresentar ao cliente.
