# Assets reais do espaço — Clínica Arvorear

Estrutura pensada para Next.js em `public/assets/arvorear/espaco`.

## Como usar

Extraia o ZIP na raiz do projeto. O caminho final deve ficar:

```txt
public/assets/arvorear/espaco/
```

Use os arquivos `.webp` no site. Eles já estão convertidos e otimizados para web.

## Melhores usos para o site

- **Hero / seção O Espaço:** `/assets/arvorear/espaco/hero/hero-recepcao-real-16x9.webp`
- **Contato / localização:** `/assets/arvorear/espaco/hero/hero-fachada-real-wide.webp`
- **Card premium da recepção:** `/assets/arvorear/espaco/hero/card-recepcao-logo-4x5.webp`
- **Galeria real:** pastas `recepcao`, `ambientes`, `fachada` e `detalhes`
- **Render/conceito:** pasta `conceito-3d`; use com moderação ou como apoio visual.
