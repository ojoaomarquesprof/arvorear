# Arvorear — assets reais do espaço

Extraia este ZIP na raiz do projeto Next.js.

Depois da extração, a estrutura deve ficar assim:

```txt
public/assets/arvorear/espaco/
  hero/
  recepcao/
  ambientes/
  fachada/
  detalhes/
  conceito-3d/
  thumbs/
  _docs/
  _preview/
```

O Codex deve usar preferencialmente:

- `/assets/arvorear/espaco/hero/hero-recepcao-real-16x9.webp`
- `/assets/arvorear/espaco/hero/hero-fachada-real-wide.webp`
- `/assets/arvorear/espaco/hero/card-recepcao-logo-4x5.webp`

A pasta `_docs` contém um manifesto com caminhos, alt texts e sugestões de uso.
