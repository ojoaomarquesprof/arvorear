# Assets organizados — Clínica Arvorear

Extraia este ZIP **na raiz do projeto Next.js**, na mesma pasta onde fica o `package.json`.

Depois da extração, a estrutura principal ficará assim:

```txt
public/assets/arvorear/
├─ brand/
│  ├─ logos/
│  │  ├─ png/
│  │  └─ webp/
│  ├─ banners/
│  ├─ mockups/
│  ├─ previews/
│  └─ referencias-pdf/
├─ espaco/
│  ├─ hero/
│  ├─ recepcao/
│  ├─ ambientes/
│  ├─ fachada/
│  ├─ detalhes/
│  ├─ conceito-3d/
│  ├─ thumbs/
│  ├─ _docs/
│  └─ _preview/
└─ _docs/
```
